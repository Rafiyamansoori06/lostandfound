# Lost & Found Digital Notice Board

A simple web application where users can:

- Report lost items
- Upload item images
- Search lost items
- Claim found items

---

# Features

✅ Report Lost Item  
✅ Upload Images  
✅ Search Functionality  
✅ Claim Button  
✅ Responsive Design  
✅ Appwrite Backend Integration
